<Note>
The Livestream Scheduling API is in **closed beta** — breaking changes should be expected. See the [overview](/livestream-api/scheduled-broadcasts/overview).
</Note>

Deletes a scheduled broadcast. `:id` is the alphanumeric `broadcast_id`.

**Endpoint:** `DELETE /2/broadcasts/scheduled/:id`

**OAuth 2.0 scopes:** `broadcast.read`, `broadcast.write`

For a recurring schedule, optionally pass `roll_forward=true` to apply the deletion to subsequent occurrences.

## Example

```bash
curl --request DELETE \
  --url 'https://api.x.com/2/broadcasts/scheduled/1DxLddgepQbxm' \
  --header 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```

## Response

```json
{
  "data": {
    "deleted": true
  }
}
```
