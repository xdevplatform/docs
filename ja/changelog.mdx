<Update label="2026年5月20日" description="開発者プラットフォーム" >
### console.x.com での新しい開発者サインアップ体験
[console.x.com](https://console.x.com) で開発者アカウントにサインアップし、アプリと認証情報を管理できるようになりました。Ads API の[ステップバイステップガイド](/x-ads-api/getting-started/step-by-step-guide)は、新しいサインアップフローを反映するように更新されました。

### ドキュメントが日本語、スペイン語、ポルトガル語、韓国語で利用可能に
X 開発者プラットフォームのドキュメントが、新たに 4 言語で利用可能になりました。ドキュメントの言語スイッチャーを使って、お好みの言語でコンテンツをお読みいただけます。まずは Ads API の[ステップバイステップガイド](/x-ads-api/getting-started/step-by-step-guide)からご覧ください。
</Update>
