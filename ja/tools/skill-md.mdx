[`skill.md`](https://docs.x.com/skill.md) ファイルは [agentskills.io 仕様](https://agentskills.io/specification) に準拠し、AI エージェントが X API で*何ができるか*を記述します。[`llms.txt`](/tools/llms-txt) がページのディレクトリであるのに対し、`skill.md` は機能サマリーです。特定のアクション、必須の入力、制約を列挙することで、エージェントが API をより確実に利用できるようにします。

---

## 含まれる内容

- **Capabilities** — エージェントが達成できること (投稿の検索、投稿の作成、ユーザー管理など)
- **Skills** — カテゴリ別に整理された、必須パラメーター付きの特定アクション
- **Workflows** — 一般的なタスクを実行するためのステップバイステップの手順
- **Context** — 認証、レート制限、アーキテクチャに関する背景情報

```bash
# skill ファイルを取得
curl https://docs.x.com/skill.md
```

## ディスカバリーエンドポイント

エージェントは well-known エンドポイントを介してプログラム的に skill ファイルを発見できます。

```bash
# ディスカバリーエンドポイント (agent-skills 0.2.0 仕様)
curl https://docs.x.com/.well-known/agent-skills/index.json

# 従来のディスカバリー形式
curl https://docs.x.com/.well-known/skills/index.json
```

[skills CLI](https://www.npmjs.com/package/skills) をサポートする任意のエージェントに X API の機能を追加できます。

```bash
npx skills add https://docs.x.com
```
