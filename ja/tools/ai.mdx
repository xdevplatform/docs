---

## AI ツールから X API を使う

AI エージェントに X API エンドポイントを直接呼び出す能力を与えます。

**MCP サーバー** — XMCP は 200 以上の X API エンドポイントを呼び出し可能な MCP ツールとして公開します。Docs MCP を使うと、エージェントがドキュメントをその場で検索・参照できます。

**OpenAPI 仕様** — 機械可読な API 定義 (https://api.x.com/2/openapi.json)。クライアントの自動生成やエージェントへの取り込みに利用できます。

| リソース | 用途 | URL |
|:---------|:------------|:----|
| **X MCP** | ホスト型 MCP サーバー — エージェントは検索、ユーザー参照、ブックマーク管理などが可能 | [docs.x.com/tools/mcp](/tools/mcp) |
| **OpenAPI 仕様** | コード生成やエージェントツール向けの機械可読な API 定義 | [api.x.com/2/openapi.json](https://api.x.com/2/openapi.json) |

## AI ツールから X API のドキュメントを読む

AI エージェントに X API の仕組みに関するコンテキストを与えます。

**llms.txt / llms-full.txt** — LLM に取り込ませるためのドキュメントインデックスおよび全文の Markdown ファイル。

**skill.md** — 利用可能なアクション、パラメーター、制約をエージェントに正確に伝える機能サマリー (agentskills.io)。

**Docs MCP** — MCP を通じて AI アシスタントから任意のドキュメントページを検索・参照できます。

| リソース | 用途 | URL |
|:---------|:------------|:----|
| **llms.txt** | ドキュメントインデックス — ページタイトル、URL、説明 | [docs.x.com/llms.txt](https://docs.x.com/llms.txt) |
| **llms-full.txt** | フルコンテキスト読み込み用の全ドキュメントを 1 ファイルに集約 | [docs.x.com/llms-full.txt](https://docs.x.com/llms-full.txt) |
| **skill.md** | 機能サマリー — アクション、入力、制約 | [docs.x.com/skill.md](https://docs.x.com/skill.md) |
| **Docs MCP** | MCP サーバー — エージェントがドキュメントページをその場で検索・参照可能 | [docs.x.com/mcp](https://docs.x.com/mcp) |
