**llms.txt** — すべてのドキュメントページ (タイトル、URL、短い説明) の構造化されたインデックスです。AI ツールに利用可能なコンテンツの地図を提供します。

**llms-full.txt** — 最大限のコンテキストを提供するために、ドキュメント全体を 1 つの Markdown ファイルにまとめたものです。

[`llms.txt` 標準](https://llmstxt.org) は AI 向けのサイトマップのようなものです。`sitemap.xml` が検索エンジンを支援するのと同様に、LLM がドキュメント構造を理解し、関連コンテンツを見つけるのを助けます。

| ファイル | 内容 | 適した用途 |
|:-----|:-----------------|:---------|
| [`llms.txt`](https://docs.x.com/llms.txt) | キュレートされたルートインデックス + セクションインデックスへのリンク | 概要把握やエージェントのエントリーポイント |
| [`x-api/llms.txt`](https://docs.x.com/x-api/llms.txt) | 370 以上の X API v2 リファレンスページ | Posts、Users、DMs、Streams、Compliance など |
| [`enterprise-api/llms.txt`](https://docs.x.com/enterprise-api/llms.txt) | Enterprise、Account Activity、GNIP、Webhooks | 過去データおよびリアルタイムの Enterprise データ |
| [`x-ads-api/llms.txt`](https://docs.x.com/x-ads-api/llms.txt) | Ads API (キャンペーン、クリエイティブ、オーディエンス、アナリティクス) | 広告と計測 |
| [`xdks/python/llms.txt`](https://docs.x.com/xdks/python/llms.txt) + [`xdks/typescript/llms.txt`](https://docs.x.com/xdks/typescript/llms.txt) | SDK クライアントおよびモデルの完全リファレンス | Python および TypeScript/JavaScript デベロッパー |
| [`llms-full.txt`](https://docs.x.com/llms-full.txt) | 全ドキュメントを 1 つの Markdown ファイルとして提供 | 深い推論のための最大コンテキスト |

これらのファイル (加えて `/x-api/llms.txt`、`/enterprise-api/llms.txt`、`/x-ads-api/llms.txt`、および XDK 配下のセクション別インデックス) はルートおよび `/.well-known/` 配下で利用できます。すべてのドキュメントページはクリーンな Markdown を得るための `.md` サフィックスをサポートしています (例: `https://docs.x.com/x-api/posts/get-post-by-id.md`)。

## 使い方

**エージェントには強く推奨:** まず [AGENTS.md](https://docs.x.com/AGENTS.md) (または AGENT.md) を読み、明示的な利用ガイダンスを確認してください。

ほとんどの AI ツールは URL を直接受け付けます。URL を指定すれば、ツールがコンテンツを取得・解析します。

- **Grok**: `https://docs.x.com/llms-full.txt` をチャットに貼り付け、X API に関する質問をします
- **Cursor / Windsurf**: `https://docs.x.com/llms-full.txt` をプロジェクトのドキュメントコンテキストとして追加します
- **カスタムエージェント**: ファイルをプログラム的に取得し、システムプロンプトやコンテキストウィンドウに含めます

```bash
# ドキュメントインデックスを取得
curl https://docs.x.com/llms.txt

# ドキュメント全体を取得
curl https://docs.x.com/llms-full.txt

# 個別のページを Markdown として取得
curl https://docs.x.com/tools/llms-txt.md
```
