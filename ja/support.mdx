## トラブルシューティングのヒント

<AccordionGroup>
  <Accordion title="401 Unauthorized エラーが発生する" icon="lock">
    - API キーとトークンが正しいかをよく確認してください
    - エンドポイントに適した認証方法 (OAuth 1.0a vs OAuth 2.0) を使用していることを確認してください
    - アプリに必要な権限 (読み取り、書き込み、DM アクセス) があることを確認してください
    - トークンが有効期限切れでないかを確認してください — 必要に応じて再生成してください
  </Accordion>
  
  <Accordion title="レート制限に達している" icon="gauge">
    - API レスポンスの `x-rate-limit-*` ヘッダーを確認して使用状況をモニタリングしてください
    - 429 エラーを受信した場合は指数バックオフを実装してください
    - API コールを減らすために可能な場所でレスポンスをキャッシュしてください
    - より高い上限のアクセスティアへのアップグレードを検討してください
    - 詳細については [レート制限ガイド](/x-api/fundamentals/rate-limits) を参照してください
  </Accordion>
  
  <Accordion title="アプリが停止された" icon="ban">
    - 停止の原因として考えられる事項を理解するために [デベロッパーガイドライン](/developer-guidelines) を確認してください
    - 停止について X からの連絡がないかメールを確認してください
    - [Policy Support フォーム](https://help.x.com/forms/platform) を通じて申し立てを提出してください
    - よくある原因: 自動いいね、迷惑な DM/メンション、スクレイピング、レート制限の悪用
  </Accordion>
  
  <Accordion title="より高い API アクセスまたは上限が必要" icon="arrow-up">
    - 適切なプランを見つけるために [料金ティア](/x-api/getting-started/pricing) を確認してください
    - エンタープライズのニーズには、[Enterprise API Interest フォーム](/forms/enterprise-api-interest) を送信してください
    - アクセスのアップグレードには、[Use Case Upgrade フォーム](/forms/use-case/upgrade) を使用してください
  </Accordion>
  
  <Accordion title="必要なエンドポイントが見つからない" icon="magnifying-glass">
    - 完全な [API リファレンス](/x-api/introduction) を参照してください
    - その機能が v2 にまだない [v1.1 エンドポイント](/x-api/migrate/overview) に存在するか確認してください
    - 不明な場合は [デベロッパーフォーラム](https://devcommunity.x.com) で質問してください
  </Accordion>
</AccordionGroup>
