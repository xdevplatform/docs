[Developer Console](https://console.x.com) は、X API アクセスを管理するためのセントラルハブです。アプリの作成、認証情報の生成、使用状況のモニタリング、請求の管理がすべて一箇所で行えます。

<Button href="https://console.x.com">Developer Console を開く</Button>

---

## できること

<CardGroup cols={2}>
  <Card title="アプリの作成と管理" icon="grid-2">
    アプリを設定して API 認証情報を取得します。認証、権限、コールバック URL を設定します。
  </Card>
  <Card title="使用状況とコストのモニタリング" icon="chart-line">
    API 使用状況をリアルタイムで追跡します。エンドポイントごとのコストを表示し、クレジット残高を管理します。
  </Card>
  <Card title="認証情報の生成" icon="key">
    アプリ用の API キー、アクセストークン、OAuth 認証情報を作成します。
  </Card>
  <Card title="クレジットの購入" icon="credit-card">
    従量課金制の請求のためにクレジットを購入します。サブスクリプションやコミットメントは不要です。
  </Card>
</CardGroup>

## はじめに

<Steps>
  <Step title="デベロッパーアカウントにサインアップ">
    [console.x.com](https://console.x.com) にアクセスし、X アカウントでサインインします。デベロッパー契約に同意します。
  </Step>
  <Step title="最初のアプリを作成">
    **New App** をクリックし、名前と説明を入力します。これにより API 認証情報が生成されます。
  </Step>
  <Step title="認証情報を保存">
    API Key、API Secret、Access Token をコピーして安全に保存してください。これらは再表示されません。
  </Step>
  <Step title="構築を始める">
    認証情報を使用して API リクエストを認証します。[クイックスタートガイド](/x-api/getting-started/make-your-first-request) をご覧ください。
  </Step>
</Steps>

<Warning>
**認証情報を安全に保存してください。** API キーとトークンは、生成時に一度だけ表示されます。パスワードマネージャーや安全な保管庫を使用してください。紛失した場合は再生成する必要があり、古い認証情報は無効になります。
</Warning>

## 請求とクレジット

X API は、クレジットベースのシステムによる従量課金制を採用しています:

| 機能 | 説明 |
|:--------|:------------|
| **月額料金なし** | 使った分だけ支払い — サブスクリプション不要 |
| **クレジットベース** | クレジットを事前に購入し、API を使用すると控除されます |
| **リアルタイム追跡** | コンソールダッシュボードで使用状況とコストをモニタリング |
| **エンドポイント別料金** | 異なるエンドポイントには異なるコストがあります |
| **重複排除** | 同じリソースが 24 時間以内に 2 回リクエストされた場合は 1 回のみ課金 |

現在の料金確認とクレジットの購入は [Developer Console](https://console.x.com) で行えます。

[料金の詳細はこちら →](/x-api/getting-started/pricing)

## チーム管理

<Note>
チーム管理は **エンタープライズアカウント** のみで利用可能です。
</Note>

エンタープライズアカウントでは、コラボレーションのためにチームメンバーを招待できます:

| ロール | 機能 |
|:-----|:-------------|
| **管理者** | フルアクセス: アプリ、請求、チームメンバー、ロールの管理 |
| **開発者** | 自身のアプリを管理、チームアプリへの読み取り専用アクセス |

チームメンバーを招待するには、コンソールの Team ページに移動し、その X ハンドルを入力します。
