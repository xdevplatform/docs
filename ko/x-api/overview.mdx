X API는 X의 공개 대화에 프로그래밍 방식으로 액세스할 수 있게 해줍니다. 아래의 모든 엔드포인트는 [사용량 기반](/x-api/introduction) 요금제에서 사용할 수 있습니다. Enterprise 전용으로 표시된 엔드포인트는 [Enterprise](/enterprise-api/introduction) 요금제가 필요합니다.
