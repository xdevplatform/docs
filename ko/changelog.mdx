업데이트를 구독하려면 [**@API**](https://x.com/api)의 [**"알림 켜기"**](https://help.x.com/en/managing-your-account/notifications-on-mobile-devices#:~:text=In%20the%20top%20menu,%20you,you%20would%20like%20to%20receive)를 활성화하세요.

<Update label="2026년 5월 20일" description="개발자 플랫폼" >
### console.x.com의 새로운 개발자 가입 환경
이제 [console.x.com](https://console.x.com)에서 개발자 계정에 가입하고 앱과 자격 증명을 관리할 수 있습니다. Ads API [단계별 가이드](/x-ads-api/getting-started/step-by-step-guide)가 새로운 가입 흐름을 반영하도록 업데이트되었습니다.

### 일본어, 스페인어, 포르투갈어, 한국어로 문서 제공
X 개발자 플랫폼 문서가 네 가지 언어로 추가 제공됩니다. 문서의 언어 전환기를 사용해 선호하는 언어로 콘텐츠를 읽어 보세요. Ads API [단계별 가이드](/x-ads-api/getting-started/step-by-step-guide)부터 시작할 수 있습니다.
</Update>
