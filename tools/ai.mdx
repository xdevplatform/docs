Give your AI agent the ability to call X API endpoints directly.

## Read X API docs from AI tools

Give your AI agent context on how the X API works.

<CardGroup cols={2}>
  <Card title="llms.txt" icon="file-lines" href="/tools/llms-txt">
    Documentation index and full-content files for LLMs to ingest.
  </Card>
  <Card title="skill.md" icon="wand-magic-sparkles" href="/tools/skill-md">

  <Card title="Docs MCP" icon="book-open" href="/tools/mcp#docs-mcp--documentation-search">

| Resource | What it does | URL |
|:---------|:------------|:----|
