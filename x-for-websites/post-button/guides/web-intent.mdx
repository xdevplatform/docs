A Post Web Intent makes it easy for your site visitors to compose and post a Post to their audience from a link on your webpage. Sites may configure Post text and hashtags, pass a URL, and identify X accounts related to the page.

## Web Intent URL

```
https://x.com/intent/tweet
```

## Query parameters

| Parameter     | Description                                                                 | Example Value                                      |
|---------------|-----------------------------------------------------------------------------|----------------------------------------------------|
| `text`        | Pre-populated UTF-8 and URL-encoded Post text. The passed text will appear pre-selected for a X user to delete or edit before posting. The length of your passed Post text should not exceed 280 characters when combined with any passed hashtags, via, or url parameters. Passed Post text which causes the Post to exceed 280 characters in length will require additional editing by a X user before they can successfully post. | `Hello%20World`                                    |
| `url`         | A fully-qualified URL with a HTTP or HTTPS scheme, [URL-encoded](http://en.wikipedia.org/wiki/Percent-encoding). The provided URL will be shortened by X's t.co to the number of characters specified by short_url_length. | `https%3A%2F%2Fexample.com%2F`                     |
| `hashtags`    | Allow easy discovery of Posts by topic by including a comma-separated list of hashtag values without the preceding # character. | `nature,sunset`                                    |
| `via`         | A X username to associate with the Post, such as your site’s X account. The provided username will be appended to the end of the Post with the text “via @username”. A logged-out X user will be encouraged to sign-in or join X to engage with the via account’s Posts. The account may be suggested as an account to follow after the user posts a Post. | `xdevelopers`                                       |
| `related`     | Suggest additional X usernames related to the Post as comma-separated values. X may suggest these accounts to follow after the user posts their Post. You may provide a brief description of how the account relates to the Post with a URL-encoded comma and text after the username. | `x%3AX%20News,xapi%3AX%20API%20News` |
| `in_reply_to` | The Post ID of a parent Post in a conversation, such as the initial Post from your site or author account. | `525001166233403393`                               |
