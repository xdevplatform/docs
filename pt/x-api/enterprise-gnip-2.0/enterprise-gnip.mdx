Nossas APIs enterprise oferecem o mais alto nível de acesso e confiabilidade para quem depende dos dados do X. Ideais à medida que você escala além do premium e precisa de acesso mais confiável, pacotes personalizados ou contratos de longo prazo. O acesso às APIs enterprise inclui gerentes de conta dedicados e suporte técnico personalizado.

Se você identificou que o plano de autoatendimento não é suficiente para atender às suas necessidades, pode solicitar acesso aos produtos enterprise no link a seguir:

<Button href="https://docs.x.com/forms/enterprise-api-interest">
  Solicitar acesso
</Button>

## Endpoints disponíveis

| Tempo real | [PowerTrack API](/x-api/enterprise-gnip-2.0/powertrack-api) | Fornece cobertura completa de Posts públicos em tempo real, filtráveis usando uma ampla variedade de operadores e regras. |
| :--- | :--- | :--- |
| [Decahose API](/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | Transmite uma amostra aleatória de 10% de todos os Posts públicos e suas curtidas. |
| :--- | :--- | :--- |
| [Account Activity API](/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | Inscreva-se em usuários e receba diversas atividades, incluindo Posts, mensagens diretas, curtidas, seguidores e muito mais. |
| :--- | :--- | :--- |
| Histórico | [Search API](/x-api/enterprise-gnip-2.0/fundamentals/search-api) | Pesquise os últimos 30 dias ou o arquivo completo de Posts públicos usando ferramentas avançadas de filtragem. |
| [Historical PowerTrack API](/x-api/enterprise-gnip-2.0/powertrack-api) | Filtre o arquivo completo de Posts públicos por meio de um processo econômico de acesso em lote. |
| Insights | [Engagement API](/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | Meça o volume de impressões e engajamentos de Posts públicos. |
| Conformidade | [Compliance Firehose API](/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | Transmite todos os eventos de conformidade. |
| Uso | [Usage API](/x-api/enterprise-gnip-2.0/fundamentals/usage) | Fornece acesso programático ao consumo de atividade entre produtos enterprise. |

## Conceitos importantes

### Console do Gnip

Você pode gerenciar seus produtos enterprise e o acesso à conta pelo [Console do Gnip](/x-api/enterprise-gnip-2.0/fundamentals/overview).


### Enriquecimentos

Adicionamos metadados ao payload de resposta das APIs premium para desenvolvedores que estão em um nível pago. Saiba mais sobre URLs expandidas, dados de enquetes e geolocalização de perfil visitando nossa [seção de enriquecimentos](/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments).


### Ferramentas avançadas de filtragem

A Premium Search API disponibiliza um conjunto avançado de operadores que não estão disponíveis com o endpoint de busca padrão. Muitos desses operadores estão disponíveis para usuários do sandbox gratuito, mas você terá um conjunto de operadores mais robusto se usar um nível pago. Saiba mais visitando nossa [seção de regras e filtros](/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries).
