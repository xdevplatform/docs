## Principais recursos

<Tabs>
  <Tab title="Acesso a dados">
    ### Objetos de dados ricos
    
    Acesse dados detalhados e estruturados de posts, usuários, mídia e muito mais:
    
    - **Posts**: texto completo, métricas, entidades, anotações, threads de conversa
    - **Usuários**: perfis, contagens de seguidores, status de verificação
    - **Mídia**: imagens, vídeos, GIFs com metadados
    - **Enquetes**: opções e contagens de votos
    
    Personalize as respostas com [fields](/x-api/fundamentals/fields) e [expansions](/x-api/fundamentals/expansions) para obter exatamente os dados de que precisa.
  </Tab>
  <Tab title="Streaming quase em tempo real">
    ### Filtered stream
    
    Receba posts quase em tempo real conforme são publicados. Defina até 1.000 regras de filtragem para receber apenas os posts correspondentes.
    
    ```bash
    # Add a rule
    curl -X POST "https://api.x.com/2/tweets/search/stream/rules" \
      -H "Authorization: Bearer $TOKEN" \
      -d '{"add": [{"value": "from:xdevelopers"}]}'
    
    # Connect to stream
    curl "https://api.x.com/2/tweets/search/stream" \
      -H "Authorization: Bearer $TOKEN"
    ```
    
    [Saiba mais sobre filtered stream →](/x-api/posts/filtered-stream/introduction)
  </Tab>
  <Tab title="Pesquisa e análise">
    ### Pesquisa em arquivo completo
    
    Pesquise todo o histórico de posts públicos — desde 2006. Construa consultas com operadores para usuários, palavras-chave, datas e muito mais.
    
    ```bash
    curl "https://api.x.com/2/tweets/search/all?query=AI%20lang:en" \
      -H "Authorization: Bearer $TOKEN"
    ```
    
    ### Métricas
    
    Acesse métricas de engajamento, incluindo impressões, curtidas, reposts, respostas e visualizações de vídeo.
    
    [Saiba mais sobre pesquisa →](/x-api/posts/search/introduction)
  </Tab>
</Tabs>

## Início rápido

<Steps>
  <Step title="Crie uma conta de desenvolvedor">
    Cadastre-se em [console.x.com](https://console.x.com) e crie um app.
  </Step>
  <Step title="Obtenha suas credenciais">
    Gere seu Bearer Token para solicitações app-only.
  </Step>
  <Step title="Faça uma solicitação">
    Tente consultar um usuário:
    
    ```bash
    curl "https://api.x.com/2/users/by/username/xdevelopers" \
      -H "Authorization: Bearer $BEARER_TOKEN"
    ```
  </Step>
</Steps>
