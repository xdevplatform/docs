A API do X oferece acesso programático à conversa pública do X. Recupere posts, analise tendências, crie integrações e desenvolva novas experiências na plataforma.

---

## O que você pode fazer

| Recurso | Descrição |
|:-----------|:------------|
| **Ler posts** | Pesquise, consulte e transmita posts em tempo real |
| **Publicar conteúdo** | Crie posts, respostas e threads |
| **Gerenciar usuários** | Consulte usuários e gerencie quem você segue, bloqueia e silencia |
| **Analisar dados** | Acesse métricas, tendências e análises de engajamento |
| **Criar integrações** | Envie DMs, gerencie listas e interaja com Spaces |

## Versões da API

<Tabs>
  <Tab title="X API v2 (Recommended)">
    A versão atual da API do X, com recursos modernos e preços flexíveis.
    
    **Por que usar a v2:**
    - Preços de pagamento por uso
    - Formato de resposta JSON moderno
    - [Fields](/x-api/fundamentals/fields) e [expansions](/x-api/fundamentals/expansions) flexíveis

    - Todos os novos endpoints e recursos
    
    **Começando:**
    1. Cadastre-se em [console.x.com](https://console.x.com)

    3. [Faça sua primeira solicitação](/x-api/getting-started/make-your-first-request)
  </Tab>
  <Tab title="X API v1.1 (Legacy)">

    - Alguns endpoints de upload de mídia
    - Streaming legado (descontinuado)
    - Alguns endpoints especializados
    
    **Migração para a v2:**
    Consulte o [guia de migração](/x-api/migrate/overview) para mapeamento de endpoints e alterações no formato dos dados.
  </Tab>
</Tabs>

## Destaques da v2

<Accordion title="Fields e expansions">
Solicite apenas os dados de que você precisa. Use os parâmetros `fields` para selecionar atributos específicos e `expansions` para incluir objetos relacionados.

```bash
curl "https://api.x.com/2/tweets/123?tweet.fields=created_at,public_metrics&expansions=author_id&user.fields=username" \
  -H "Authorization: Bearer $TOKEN"
```

[Saiba mais sobre fields →](/x-api/fundamentals/fields)
</Accordion>

<Accordion title="Anotações de posts">
Os posts incluem anotações semânticas que identificam pessoas, lugares, produtos e tópicos. Filtre streams e pesquisas por tópico.

[Saiba mais sobre anotações →](/x-api/fundamentals/post-annotations)
</Accordion>

<Accordion title="Métricas de engajamento">
Acesse métricas públicas (curtidas, reposts, respostas) e métricas privadas (impressões, cliques) dos seus próprios posts.

[Saiba mais sobre métricas →](/x-api/fundamentals/metrics)
</Accordion>

<Accordion title="Rastreamento de conversas">
Reconstrua threads de conversa inteiros usando `conversation_id`. Acompanhe as respostas em todo o thread.

[Saiba mais sobre rastreamento de conversas →](/x-api/fundamentals/conversation-id)
</Accordion>

<Accordion title="Histórico de edições">
Acesse o histórico de edições dos posts, incluindo todas as versões anteriores e os metadados de edição.

[Saiba mais sobre edição de posts →](/x-api/fundamentals/edit-posts)
</Accordion>

## Preços

A X API v2 usa o modelo de **pagamento por uso**:

| Benefício | Descrição |
|:--------|:------------|
| **Sem assinaturas** | Pague apenas pelo que usar |
| **Baseado em créditos** | Compre créditos que são debitados por solicitação |
| **Acompanhamento em tempo real** | Monitore o uso no Developer Console |
| **Deduplicação** | O mesmo recurso solicitado duas vezes em 24 horas é cobrado apenas uma vez |

[Ver detalhes de preços →](/x-api/getting-started/pricing)
