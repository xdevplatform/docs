A Enterprise API oferece o mais alto nível de acesso aos dados do X. Inclui tudo o que está disponível na X API padrão, além de endpoints exclusivos de alto volume, operadores de semantic embedding para Filtered Stream, limites de taxa personalizados e gerenciamento de conta dedicado para organizações que precisam de dados do X em escala.

---

## O que você pode fazer

Tudo na X API padrão, além de:

| Capacidade | Descrição |
|:-----------|:------------|
| **Streaming do firehose completo** | Acesse 100% dos posts públicos em tempo real com volume streams |
| **Streaming de todas as curtidas** | Streams completos e amostrados de curtidas em tempo real |
| **Streaming filtrado avançado** | Powerstream (filtragem por palavra-chave de baixa latência) |
| **Operadores de semantic embedding** | Combine posts por significado (não apenas palavras-chave) usando `embedding:` no Filtered Stream (Enterprise + tier Embedding) |
| **Análise profunda de engajamento** | Métricas de engajamento de posts e mídia em escala |
| **Eventos de account activity** | Assinaturas em tempo real para eventos de usuário (posts, DMs, curtidas, follows) |
| **Entrega via webhook** | Receba dados de filtered stream via webhooks |
| **Limites de taxa personalizados** | Limites elevados sob medida para suas necessidades de throughput |
| **Suporte dedicado** | Gerente de conta nominal e resolução prioritária de problemas |

## Destaques da API

<Accordion title="Fields e expansions">
Solicite apenas os dados de que precisa. Use os parâmetros `fields` para selecionar atributos específicos e `expansions` para incluir objetos relacionados.

```bash
curl "https://api.x.com/2/tweets/123?tweet.fields=created_at,public_metrics&expansions=author_id&user.fields=username" \
  -H "Authorization: Bearer $TOKEN"
```

[Saiba mais sobre fields →](/x-api/fundamentals/fields)
</Accordion>

<Accordion title="Anotações de posts">
Os posts incluem anotações semânticas que identificam pessoas, lugares, produtos e tópicos. Filtre streams e pesquisas por tópico.

[Saiba mais sobre anotações →](/x-api/fundamentals/post-annotations)
</Accordion>

<Accordion title="Métricas de engajamento">
Clientes Enterprise têm acesso a endpoints dedicados de métricas de engajamento para posts e mídia, oferecendo análises profundas em escala, além do que as métricas públicas padrão oferecem.

[Saiba mais sobre métricas →](/x-api/fundamentals/metrics)
</Accordion>

<Accordion title="Rastreamento de conversas">
Reconstrua threads de conversa inteiros usando `conversation_id`. Acompanhe as respostas em todo o thread.

[Saiba mais sobre rastreamento de conversas →](/x-api/fundamentals/conversation-id)
</Accordion>

<Accordion title="Histórico de edições">
Acesse o histórico de edições dos posts, incluindo todas as versões anteriores e os metadados de edição.

[Saiba mais sobre edição de posts →](/x-api/fundamentals/edit-posts)
</Accordion>

## Preços

Os planos Enterprise são personalizados de acordo com as necessidades da sua organização. Os preços baseiam-se no seu volume de dados, uso de endpoints e requisitos de suporte.

| Recurso | Detalhes |
|:--------|:--------|
| **Pacotes personalizados** | Adaptados às suas necessidades específicas de dados e throughput |
| **Contratos de longo prazo** | Preços previsíveis com acordos de uso comprometido |
| **Sem limite de leitura de posts** | Volumes personalizados ou ilimitados de leitura de posts |
| **Suporte dedicado** | Incluído em todos os planos Enterprise |
