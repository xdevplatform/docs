## Referência rápida

### Estrutura de resposta

Todas as respostas v2 seguem esta estrutura:

```json
{
  "data": { ... },      // Primary object(s)
  "includes": { ... },  // Expanded objects (if requested)
  "meta": { ... },      // Pagination info
  "errors": [ ... ]     // Partial errors (if any)
}
```

### Parâmetros comuns

| Parâmetro | Uso |
|:----------|:----|
| `tweet.fields` | Solicitar campos específicos de post |
| `user.fields` | Solicitar campos específicos de usuário |
| `expansions` | Incluir objetos relacionados |
| `max_results` | Limitar resultados por página |
| `pagination_token` | Obter próxima/página anterior |

### Métodos de autenticação

| Método | Caso de uso |
|:-------|:---------|
| Bearer Token | Dados públicos somente leitura |
| OAuth 2.0 | Ações de usuário com escopos granulares |
| OAuth 1.0a | Ações de usuário (legado, permissões completas) |

[Guia completo de autenticação →](/resources/fundamentals/authentication/overview)
