## Dicas de solução de problemas

<AccordionGroup>
  <Accordion title="Estou recebendo erros 401 Unauthorized" icon="lock">
    - Verifique novamente se suas API keys e tokens estão corretos
    - Garanta que está usando o método de autenticação correto para o endpoint (OAuth 1.0a vs OAuth 2.0)
    - Verifique se seu app tem as permissões necessárias (read, write, acesso a DM)
    - Verifique se seus tokens não expiraram — regenere se necessário
  </Accordion>
  
  <Accordion title="Estou atingindo rate limits" icon="gauge">
    - Verifique os headers `x-rate-limit-*` nas respostas da API para monitorar seu uso
    - Implemente backoff exponencial quando receber erros 429
    - Faça cache das respostas quando possível para reduzir chamadas à API
    - Considere fazer upgrade do seu tier de acesso para limites maiores
    - Veja o [guia de Rate Limits](/x-api/fundamentals/rate-limits) para informações detalhadas
  </Accordion>
  
  <Accordion title="Meu app foi suspenso" icon="ban">
    - Revise as [Diretrizes para Desenvolvedores](/developer-guidelines) para entender o que pode ter causado a suspensão
    - Verifique seu e-mail para qualquer comunicação do X sobre a suspensão
    - Envie um recurso através do [formulário de Policy Support](https://help.x.com/forms/platform)
    - Causas comuns: curtidas automatizadas, DMs/menções não solicitadas, scraping, abuso de rate limit
  </Accordion>
  
  <Accordion title="Preciso de acesso ou limites maiores da API" icon="arrow-up">
    - Revise os [tiers de preços](/x-api/getting-started/pricing) para encontrar o plano certo
    - Para necessidades enterprise, envie o [formulário de Interesse na Enterprise API](/forms/enterprise-api-interest)
    - Para upgrades de acesso, use o [formulário de Upgrade de Caso de Uso](/forms/use-case/upgrade)
  </Accordion>
  
  <Accordion title="Não consigo encontrar o endpoint que preciso" icon="magnifying-glass">
    - Navegue pela [Referência da API](/x-api/introduction) completa
    - Verifique se a funcionalidade existe nos [endpoints v1.1](/x-api/migrate/overview) que talvez ainda não estejam em v2
    - Pergunte nos [Fóruns do Desenvolvedor](https://devcommunity.x.com) se estiver em dúvida
  </Accordion>
</AccordionGroup>
