O [Developer Console](https://console.x.com) é seu hub central para gerenciar o acesso à X API. Crie apps, gere credenciais, monitore o uso e gerencie a cobrança — tudo em um só lugar.

<Button href="https://console.x.com">Abrir Developer Console</Button>

## Começando

<Steps>
  <Step title="Cadastre-se para uma conta de desenvolvedor">
    Acesse [console.x.com](https://console.x.com) e faça login com sua conta X. Aceite o Acordo do Desenvolvedor.
  </Step>
  <Step title="Crie seu primeiro app">
    Clique em **New App** e forneça um nome e descrição. Isso gera suas credenciais de API.
  </Step>
  <Step title="Salve suas credenciais">
    Copie e armazene com segurança sua API Key, API Secret e Access Tokens. Eles não serão exibidos novamente.
  </Step>
  <Step title="Comece a desenvolver">
    Use suas credenciais para autenticar solicitações de API. Confira o [guia de início rápido](/x-api/getting-started/make-your-first-request).
  </Step>
</Steps>

<Warning>
**Armazene as credenciais com segurança.** API keys e tokens são exibidos apenas uma vez quando gerados. Use um gerenciador de senhas ou cofre seguro. Se forem perdidos, você precisará regenerá-los, o que invalida as credenciais antigas.
</Warning>

## Cobrança e créditos

A X API usa preços de pagamento por uso com um sistema baseado em créditos:

| Recurso | Descrição |
|:--------|:------------|
| **Sem mensalidades** | Pague apenas pelo que usar — sem assinaturas |
| **Baseado em créditos** | Compre créditos antecipadamente, descontados conforme você usa a API |
| **Acompanhamento em tempo real** | Monitore uso e custos no painel do console |
| **Preços por endpoint** | Endpoints diferentes têm custos diferentes |
| **Deduplicação** | Um mesmo recurso solicitado duas vezes em 24 horas é cobrado apenas uma vez |

Visualize os preços atuais e compre créditos no [Developer Console](https://console.x.com).

## Gerenciamento de equipe

<Note>
O gerenciamento de equipe está disponível apenas para **contas Enterprise**.
</Note>

Contas Enterprise podem convidar membros da equipe para colaborar:

| Função | Capacidades |
|:-----|:-------------|
| **Administrator** | Acesso total: gerenciar apps, cobrança, membros da equipe e funções |
| **Developer** | Gerenciar seus próprios apps, acesso somente leitura aos apps da equipe |

Para convidar um membro da equipe, navegue até a página Team no seu console e insira o handle do X dele.
