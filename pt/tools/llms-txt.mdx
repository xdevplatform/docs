**llms.txt** — um índice estruturado de todas as páginas da documentação (títulos, URLs, descrições curtas). Fornece às ferramentas de IA um mapa do que está disponível.

**llms-full.txt** — a documentação completa em um único arquivo Markdown para o máximo de contexto.

O [padrão `llms.txt`](https://llmstxt.org) funciona como um sitemap para IA. Ele ajuda os LLMs a entender a estrutura da sua documentação e a encontrar conteúdo relevante — de forma parecida com o que o `sitemap.xml` faz pelos mecanismos de busca.

| Arquivo | O que contém | Melhor para |
|:-----|:-----------------|:---------|
| [`llms.txt`](https://docs.x.com/llms.txt) | Índice raiz curado + links para índices de seção | Visão geral rápida e ponto de entrada para agentes |
| [`x-api/llms.txt`](https://docs.x.com/x-api/llms.txt) | Mais de 370 páginas de referência da X API v2 | Posts, usuários, DMs, streams, compliance etc. |
| [`enterprise-api/llms.txt`](https://docs.x.com/enterprise-api/llms.txt) | Enterprise, Account Activity, GNIP, Webhooks | Dados enterprise históricos + em tempo real |
| [`x-ads-api/llms.txt`](https://docs.x.com/x-ads-api/llms.txt) | Ads API (campanhas, criativos, audiências, analytics) | Publicidade e mensuração |
| [`xdks/python/llms.txt`](https://docs.x.com/xdks/python/llms.txt) + [`xdks/typescript/llms.txt`](https://docs.x.com/xdks/typescript/llms.txt) | Referência completa de cliente e modelos do SDK | Desenvolvedores em Python e TypeScript/JavaScript |
| [`llms-full.txt`](https://docs.x.com/llms-full.txt) | Documentação inteira como um único arquivo Markdown | Máximo contexto para raciocínio aprofundado |

Esses arquivos (além de índices específicos por seção em `/x-api/llms.txt`, `/enterprise-api/llms.txt`, `/x-ads-api/llms.txt` e nos XDKs) estão disponíveis na raiz e em `/.well-known/`. Toda página da documentação suporta o sufixo `.md` para Markdown limpo (exemplo: `https://docs.x.com/x-api/posts/get-post-by-id.md`).

## Como usar

**Fortemente recomendado para agentes:** comece lendo o [AGENTS.md](https://docs.x.com/AGENTS.md) (ou AGENT.md) para orientações explícitas de uso.

A maioria das ferramentas de IA aceita URLs diretamente. Basta fornecer a URL e a ferramenta fará o fetch e o parsing do conteúdo:

- **Grok**: cole `https://docs.x.com/llms-full.txt` no chat e faça perguntas sobre a X API
- **Cursor / Windsurf**: adicione `https://docs.x.com/llms-full.txt` como contexto de documentação do seu projeto
- **Agentes customizados**: baixe o arquivo programaticamente e inclua no seu system prompt ou janela de contexto

```bash
# Fetch the documentation index
curl https://docs.x.com/llms.txt

# Fetch the complete documentation
curl https://docs.x.com/llms-full.txt

# Fetch any individual page as Markdown
curl https://docs.x.com/tools/llms-txt.md
```
