Recursos para conectar ferramentas de IA à X API e à sua documentação.

---

## Use a X API a partir de ferramentas de IA

Dê ao seu agente de IA a capacidade de chamar endpoints da X API diretamente.

**Servidores MCP** — o XMCP expõe mais de 200 endpoints da X API como ferramentas MCP invocáveis. O Docs MCP permite que agentes pesquisem e leiam esta documentação em tempo real.

**Especificação OpenAPI** — definição de API legível por máquina (https://api.x.com/2/openapi.json) para gerar clientes automaticamente ou alimentar agentes.

| Recurso | O que faz | URL |
|:---------|:------------|:----|
| **X MCP** | Servidor MCP hospedado — agentes podem pesquisar, consultar usuários, gerenciar bookmarks e muito mais | [docs.x.com/tools/mcp](/tools/mcp) |
| **OpenAPI Spec** | Definição de API legível por máquina para geração de código e ferramental de agentes | [api.x.com/2/openapi.json](https://api.x.com/2/openapi.json) |

---

## Leia a documentação da X API a partir de ferramentas de IA

Dê ao seu agente de IA contexto sobre como a X API funciona.

**llms.txt / llms-full.txt** — índice da documentação e arquivos Markdown com o conteúdo completo para os LLMs consumirem.

**skill.md** — resumo de capacidades (agentskills.io) que informa aos agentes exatamente quais ações, parâmetros e restrições existem.

**Docs MCP** — pesquise e leia qualquer página da documentação diretamente do seu assistente de IA via MCP.

| Recurso | O que faz | URL |
|:---------|:------------|:----|
| **llms.txt** | Índice da documentação — títulos, URLs e descrições das páginas | [docs.x.com/llms.txt](https://docs.x.com/llms.txt) |
| **llms-full.txt** | Documentação completa em um único arquivo para carregamento de contexto integral | [docs.x.com/llms-full.txt](https://docs.x.com/llms-full.txt) |
| **skill.md** | Resumo de capacidades — ações, entradas e restrições | [docs.x.com/skill.md](https://docs.x.com/skill.md) |
| **Docs MCP** | Servidor MCP — agentes podem pesquisar e ler páginas de documentação em tempo real | [docs.x.com/mcp](https://docs.x.com/mcp) |
