## Crie para segurança e moderação

Ajude a criar conversas mais saudáveis no X.

<CardGroup cols={2}>
  <Card title="Ferramentas de moderação" icon="shield">
    Crie ferramentas para ajudar usuários a gerenciar respostas, filtrar conteúdo e proteger sua experiência.
  </Card>
  <Card title="Antiassédio" icon="ban">
    Detecte e sinalize conteúdo abusivo e ofereça ferramentas de bloqueio em massa.
  </Card>
</CardGroup>

**Endpoints relevantes:**
- [Blocks](/x-api/users/blocks/introduction) — Bloqueie e desbloqueie usuários
- [Mutes](/x-api/users/mutes/introduction) — Silencie contas e palavras-chave
- [Hide replies](/x-api/posts/hide-replies/introduction) — Oculte respostas indesejadas
- [Manage posts](/x-api/posts/manage-tweets/introduction) — Controle as configurações de resposta

## Crie com agentes de IA

Conecte ferramentas de IA à X API e crie fluxos inteligentes.

<CardGroup cols={2}>
  <Card title="Agentes baseados em MCP" icon="robot">
    Use o XMCP para dar a assistentes de IA acesso direto aos endpoints da X API — crie posts, pesquise e gerencie contas por linguagem natural.
  </Card>
  <Card title="Pipelines de dados" icon="diagram-project">
    Alimente LLMs com dados do X para análise de sentimento, geração de conteúdo e relatórios automatizados.
  </Card>
  <Card title="Assistentes de código" icon="wand-magic-sparkles">
    Dê ao Grok, Cursor ou à sua ferramenta de IA preferida o contexto completo da X API com o llms.txt para desenvolver mais rápido.
  </Card>
  <Card title="Integrações personalizadas" icon="plug">
    Crie agentes que monitoram tendências, respondem a menções ou fazem curadoria de conteúdo automaticamente.
  </Card>
</CardGroup>

**Começar:**
- [XMCP](/tools/mcp#xmcp--x-api-endpoints) — Servidor MCP para a X API
- [llms-full.txt](https://docs.x.com/llms-full.txt) — Forneça a documentação completa à sua ferramenta de IA
- [OpenAPI Spec](https://api.x.com/2/openapi.json) — Definição da API legível por máquina

[Veja todos os recursos para agentes](/tools/ai)
