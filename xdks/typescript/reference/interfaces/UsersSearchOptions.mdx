Options for search method

## Properties

<ParamField name="maxResults" type="`number`">
The maximum number of results.
</ParamField>





<ParamField name="nextToken" type="`string`">
This parameter is used to get the next 'page' of results. The value used with the parameter is pulled directly from the response provided by the API, and should not be modified.
</ParamField>





<ParamField name="userfields" type="`any`[]">
A comma separated list of User fields to display.
</ParamField>





<ParamField name="expansions" type="`any`[]">
A comma separated list of fields to expand.
</ParamField>





<ParamField name="tweetfields" type="`any`[]">
A comma separated list of Tweet fields to display.
</ParamField>





<ParamField name="requestOptions" type="[`RequestOptions`](/xdks/typescript/reference/interfaces/RequestOptions)">
Additional request options
</ParamField>
