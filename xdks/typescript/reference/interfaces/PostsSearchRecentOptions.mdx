Options for searchRecent method

## Properties

<ParamField name="startTime" type="string">
YYYY-MM-DDTHH:mm:ssZ. The oldest UTC timestamp from which the Posts will be provided. Timestamp is in second granularity and is inclusive (i.e. 12:00:01 includes the first second of the minute).
</ParamField>





<ParamField name="endTime" type="string">
YYYY-MM-DDTHH:mm:ssZ. The newest, most recent UTC timestamp to which the Posts will be provided. Timestamp is in second granularity and is exclusive (i.e. 12:00:01 excludes the first second of the minute).
</ParamField>





<ParamField name="sinceId" type="string">
Returns results with a Post ID greater than (that is, more recent than) the specified ID.
</ParamField>





<ParamField name="untilId" type="string">
Returns results with a Post ID less than (that is, older than) the specified ID.
</ParamField>





<ParamField name="maxResults" type="number">
The maximum number of search results to be returned by a request.
</ParamField>





<ParamField name="nextToken" type="string">
This parameter is used to get the next 'page' of results. The value used with the parameter is pulled directly from the response provided by the API, and should not be modified.
</ParamField>





<ParamField name="paginationToken" type="string">
This parameter is used to get the next 'page' of results. The value used with the parameter is pulled directly from the response provided by the API, and should not be modified.
</ParamField>





<ParamField name="sortOrder" type="string">
This order in which to return results.
</ParamField>





<ParamField name="tweetfields" type="string[]">
A comma separated list of Tweet fields to display.
</ParamField>





<ParamField name="expansions" type="string[]">
A comma separated list of fields to expand.
</ParamField>





<ParamField name="mediafields" type="string[]">
A comma separated list of Media fields to display.
</ParamField>





<ParamField name="pollfields" type="string[]">
A comma separated list of Poll fields to display.
</ParamField>





<ParamField name="userfields" type="string[]">
A comma separated list of User fields to display.
</ParamField>





<ParamField name="placefields" type="string[]">
A comma separated list of Place fields to display.
</ParamField>

</ParamField>
